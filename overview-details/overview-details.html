Docker Base Image for library/httpd

Approved base image by Metlife for library/httpd. It is based on the official library/httpd image from registry-1.docker.io/library/httpd.

Dockerfile Governance

Please ensure all required labels are available with accurate information in all Dockerfile(s). This includes but is not limited to:

  • EAI: EAI Code of Base Image owner

  • Product: The Name and Version of the Product available in this base image.

How to use this image

You can pull this image from JFrog Artifactory

FROM jfrog-artifactory.metlife.com/docker-metlife-base-images-virtual/library/httpd:2.4-alpine

LABEL com.metlife.image.app.description="" \
      com.metlife.image.app.dockerfile="" \
      com.metlife.image.app.source="" \
      com.metlife.image.app.maintainer="" \
      com.metlife.image.app.product.name="" \
      com.metlife.image.app.eaiCode="" \
      com.metlife.image.app.group="" \
      com.metlife.image.app.product.version=""

USER www-data
COPY dist/ /app
COPY httpd-app.conf /usr/local/apache2/conf/
